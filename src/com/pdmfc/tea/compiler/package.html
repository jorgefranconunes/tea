Provides classes used for compiling Tea programs. In order to run a Tea program two steps must be performed. First the Tea program must be compiled by a {@link com.pdmfc.tea.compiler.SCompiler SCompiler}. The result of compiling a Tea program is an {@link com.pdmfc.tea.compiler.SCode SCode} object. The second step involves the actual execution of the Tea program. This is achieved by calling the {@link com.pdmfc.tea.compiler.SCode#exec(SContext) SCode.exec(SContext)} method on the {@link com.pdmfc.tea.compiler.SCode SCode} obtained from the compilation step.
